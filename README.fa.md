# 📡 نیوزبات — ربات خبری روی Cloudflare Worker

یک ربات خبری سبک و رایگان که روی **Cloudflare Workers** اجرا می‌شه. اخبار رو از منابع مختلف دریافت می‌کنه، ترجمه می‌کنه، و به **تلگرام** و/یا **بله** ارسال می‌کنه — به‌صورت خودکار و بر اساس زمان‌بندی که خودت تنظیم می‌کنی.

بدون سرور. بدون نگهداری. کاملاً رایگان.

---

## ✨ امکانات

- دریافت خبر از فیدهای RSS و صفحات وب
- ترجمه خودکار اخبار انگلیسی به فارسی
- ارسال متن کامل خبر + تصویر + خلاصه
- جلوگیری از ارسال خبر تکراری (با KV)
- بریج پیام بین بله و تلگرام
- بیش از ۳۰ منبع از ۸ دسته‌بندی
- اجرای خودکار هر ۱۵ دقیقه
- کاملاً رایگان

---

## 🗂️ ساختار پروژه

```
newsbot/
├── worker.js          ← فایل اصلی Worker (داخل Cloudflare پیست کن)
├── sources.config.js  ← تمام منابع خبری (برای مرجع)
└── README.fa.md       ← همین فایل
```

## 🚀 راهنمای نصب (بدون نیاز به دانش کدنویسی)

### مرحله ۱ — ساخت حساب Cloudflare

به [cloudflare.com](https://cloudflare.com) برو و یه حساب رایگان بساز.

---

### مرحله ۲ — ساخت KV Namespace

KV یه حافظه ساده‌ست که ربات ازش برای یادآوری اخبار قبلاً ارسال‌شده استفاده می‌کنه.

1. داخل داشبورد Cloudflare برو به **Workers & Pages → KV**
2. روی **Create namespace** کلیک کن
3. نامش رو دقیقاً بنویس: `NEWS_KV`
4. روی **Add** کلیک کن

---

### مرحله ۳ — ساخت Worker

1. برو به **Workers & Pages → Overview**
2. روی **Create application → Create Worker** کلیک کن
3. یه نام بده (مثلاً `newsbot`)
4. روی **Deploy** کلیک کن، بعد **Edit code** رو بزن
5. **تمام کدهای پیش‌فرض رو پاک کن**
6. فایل `worker.js` این مخزن رو باز کن، همه چیز رو کپی کن، داخل ویرایشگر پیست کن
7. روی **Save and Deploy** کلیک کن

---

### مرحله ۴ — وارد کردن توکن‌ها و آیدی‌ها

بالای فایل `worker.js`، بخش `CONFIG` رو پیدا کن و پر کن:

```js
const CONFIG = {
  telegram: {
    token:      "توکن ربات تلگرامت",     // از @BotFather می‌گیری
    recipients: ["123456789"],            // آیدی چت‌های تلگرام
    admins:     ["آیدی خودت"],           // آیدی شخصی تلگرامت
  },
  bale: {
    token:      "توکن ربات بله‌ات",      // از پنل توسعه‌دهندگان بله
    recipients: ["111111", "222222"],     // آیدی چت‌های بله
  },
};
```

**چطور توکن ربات تلگرام بگیرم:**
1. داخل تلگرام `@BotFather` رو سرچ کن
2. پیام `/newbot` بفرست و مراحل رو دنبال کن
3. توکنی که می‌ده رو کپی کن

**چطور آیدی تلگرامم رو پیدا کنم:**
1. به `@userinfobot` پیام بده
2. آیدیت رو بهت می‌گه

**چطور توکن ربات بله بگیرم:**
1. داخل بله به پنل ساخت ربات برو: [dev.bale.ai](https://dev.bale.ai)
2. یه ربات بساز و توکنش رو کپی کن

---

### مرحله ۵ — اتصال KV به Worker

1. داخل تنظیمات Worker برو به **Settings → Variables**
2. به قسمت **KV Namespace Bindings** برو
3. روی **Add binding** کلیک کن
4. اسم متغیر: `NEWS_KV`
5. KV Namespace: همون `NEWS_KV` که ساختی رو انتخاب کن
6. روی **Save and Deploy** کلیک کن

---

### مرحله ۶ — تنظیم زمان‌بندی خودکار

این مرحله باعث می‌شه ربات هر ۱۵ دقیقه خودکار اجرا بشه.

1. برو به Worker → **Triggers → Cron Triggers**
2. روی **Add Cron Trigger** کلیک کن
3. بنویس: `*/15 * * * *`
4. روی **Add** کلیک کن

---

### مرحله ۷ — ثبت Webhook

این مرحله ربات‌های تلگرام و بله‌ات رو به Worker وصل می‌کنه.

1. آدرس Worker ات رو پیدا کن — شبیه این هست: `https://newsbot.yourname.workers.dev`
2. این آدرس رو داخل مرورگر باز کن:  
   `https://newsbot.yourname.workers.dev/registerWebhook`
3. باید یه پاسخ JSON ببینی که موفقیت رو تأیید کنه

فقط یه بار این کار رو بکن.

---

### مرحله ۸ — تست کردن

این آدرس رو باز کن:  
`https://newsbot.yourname.workers.dev/run`

این کار یه اجرای دستی رو شروع می‌کنه. ظرف یه دقیقه باید اولین دسته خبرها رو داخل تلگرام/بله‌ات بگیری.

وضعیت سیستم رو می‌تونی از اینجا چک کنی: `/health`

---

## 📰 تنظیم منابع خبری

فایل `sources.config.js` رو باز کن تا تمام منابع دسته‌بندی‌شده رو ببینی:

| دسته‌بندی     | مثال‌ها                                         |
|--------------|------------------------------------------------|
| `iran`       | ایران اینترنشنال، بی‌بی‌سی فارسی، رادیو فردا   |
| `world`      | Reuters، AP، BBC World، Al Jazeera            |
| `tech`       | TechCrunch، The Verge، Wired                  |
| `economy`    | Reuters Business، Bloomberg، FT               |
| `science`    | NASA، Nature، Science Daily                   |
| `ai`         | OpenAI Blog، Google AI، VentureBeat           |
| `gaming`     | IGN، Kotaku، PC Gamer                         |
| `health`     | WHO، Medical News Today                        |

منابع مورد نظرت رو از اون فایل کپی کن و داخل آرایه `SOURCES` در `worker.js` جایگذاری کن.

## ⚠️ نکات مهم

- پلن رایگان Cloudflare Workers شامل ۱۰۰,۰۰۰ درخواست در روز می‌شه — بیش از کافیه.
- KV رایگان شامل ۱۰۰,۰۰۰ خواندن و ۱,۰۰۰ نوشتن در روز هست.
- اگه منابع زیادی اضافه کردی، ممکنه به محدودیت CPU برسی. برای اطمینان، زیر ۱۵ منبع بمون.
- ترجمه از endpoint رایگان Google استفاده می‌کنه — نیازی به API key نیست.
